# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-23

### Added
- Experiment Planning skill - Transform rough ideas into comprehensive written plans
- Experiment Execution skill - Run experiments with full reproducibility tracking
- Experiment Analysis skill - Compare results vs baseline, assess outcomes
- Reproducibility Check skill - Verify experiments can be recreated exactly
- Verification Before Completion skill - Evidence before claims validation
- Experimentation Protocol reference document
- Experiment Plan Template
- Session-start hook for automatic protocol injection
- Plugin manifest for Claude Code integration

### Documentation
- Comprehensive skill documentation with When To Use, Process, Common Mistakes, and Examples
- Core experimentation rules and lifecycle documentation
- Reproducibility metadata requirements
- Outcome labeling system (WIN/LOSS/NEUTRAL/INVALID)
- Confidence level guidelines
